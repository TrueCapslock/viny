# Uva

Uva er en personlig vin- og øl-notatapp med kjellerstyring, vennefunksjoner, delte lister og personlige samlinger. Bygget med Next.js 16 App Router, Prisma + PostgreSQL (Neon), NextAuth 5 og Tailwind 4.

Norsk UI-tekst og en «øl-modus»-toggle (`useBeerMode`) som flippper hele appens språk og logo.
